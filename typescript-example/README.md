# Typescript-example

### 使用方式

clone项目

```
git clone https://github.com/juicecube/Typescript-example.git
npm install ts-node -g
```

***对应的一些代码会放在相关的文件夹***
运行使用示例：

```
ts-node xxx.ts文件
```
